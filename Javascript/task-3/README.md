# Image Gallery Lightbox

## Objective
Create an image gallery where clicking a thumbnail opens a larger version in a modal.

## Features
- Click thumbnail to open lightbox
- Dynamic image update
- Close modal via button or outside click
- Smooth transitions

## Concepts Used
- DOM event listeners
- Dynamic DOM manipulation
- CSS class toggling
- Modal UI pattern

## How to Run
Open index.html in browser.
Click images to view larger version.
